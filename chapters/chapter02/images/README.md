## 🎨 图片规范

### 推荐图片类型
- **内存模型图**: 指针、引用、堆栈布局
- **类图**: 继承、多态关系
- **数据结构图**: 数组、链表、树结构
- **算法流程图**: 排序、查找算法

### 文件格式
- **PNG**: 位图格式，适合复杂图形
- **SVG**: 矢量格式，适合图表和流程图
- **PDF**: LaTeX原生支持，高质量

## 🛠️ 添加图片
