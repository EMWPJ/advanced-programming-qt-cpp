## 🎨 图片规范

### 推荐图片类型
- **范式对比图**: 不同编程范式的特点对比
- **类图**: 设计模式结构图
- **流程图**: 算法和数据流
- **概念图**: 抽象概念的可视化

### 文件格式
- **PNG**: 适合复杂图形和截图
- **SVG**: 适合类图和流程图
- **PDF**: 矢量格式，高质量
